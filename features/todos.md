# todos

Let a user capture and track things they need to do.

## Stories

- See a list of todos.
- Create a todo.
- Toggle a todo complete / incomplete.
- Delete a todo.

## Models

### Todo

- `title` — text, required
- `done` — boolean, default false
- `createdAt` — datetime, set on create
